<h1>Tic Tac Toy Shop</h1>

<p>Tic Tac Toy Shop is a responsive and interactive e-commerce website designed for selling toys online. The platform provides a seamless shopping experience with a user-friendly interface, mobile compatibility, and essential e-commerce functionalities.</p>

<h2>Features</h2>

<h3>Home Page:</h3>

<ul>
<li>Welcome section with banner image.</li>
<li>Search functionality to find toys easily.</li>
<li>Login option for user authentication.</li>
<li>Shopping cart integration.</li>
</ul>

<h3>Product Listing Page:</h3>
<ul>

<li>Display of multiple toy products with images, names, prices, and ratings.</li>
<li>"Add to Cart" and "Buy Now" buttons.</li>
</ul>

<h3>Product Details Page:</h3>

<li>Detailed product descriptions, features, and pricing information.</li>
<li>Discount offers and availability status.</li>
<li>High-quality images of the product.</li>
<li>Call-to-action buttons for purchasing.</li>

<h3>Mobile-Friendly Design:</h3>

<li>Fully responsive layout for both desktop and mobile devices.</li>
<li>Hamburger menu for easy navigation on mobile screens.</li>

<h3>Footer Section:</h3>

<li>Links to About Us, Contact Us, Careers, and other useful pages.</li>
<li>Information on policies such as payments, shipping, and returns.</li>

<h3>Technologies Used</h3>

<h2>Frontend: HTML, CSS, JavaScript</h2>
<h2>Backend: Node.js, Express.js</h2>
<h2>Database: MongoDB</h2>
<h2>Libraries: React.js</h2>
